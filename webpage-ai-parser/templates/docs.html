Webpage -> AI-Ready Structured Data API

Local server: use http://127.0.0.1:8000/docs in your browser. Do not use https:// — there is no certificate; the page will not load.

This API takes a webpage URL, extracts readable content, and returns AI-ready structured JSON.

Endpoint

POST /parse-page

Request JSON

{
  "url": "https://example.com/article"
}

cURL Example

curl -X POST "http://127.0.0.1:8000/parse-page" ^
  -H "Content-Type: application/json" ^
  -d "{\"url\":\"https://example.com/article\"}"

Response JSON (example)

{
  "title": "Example Article",
  "summary": "Short summary of the page.",
  "sections": [
    { "heading": "Overview", "content": "..." },
    { "heading": "Details", "content": "..." }
  ],
  "key_points": ["Point 1", "Point 2"],
  "entities": ["Entity A", "Entity B"]
}
